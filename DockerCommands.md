# Docker Commands

## 1 - Gerenciamento de Imagens

### 1.1 - Baixar uma imagem do Docker Hub ou de outro registro de imagens
```
docker pull <imagem>
```
* Exemplo:
```
docker pull nginx
```

### 1.3 - Remover uma imagem do sistema
```

```
* Exemplo:
```
docker rmi nginx
```
